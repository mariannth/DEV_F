Explorando el box model y display

Esta página muestra cómo reacciona el modelo de cada con diferentes valores de display y box-sizing

Display: Block

Los elementos con display: block ocupan el ancho completo disponible y empiezan en una nueva línea

Soy un elemento block.

Display: Inline

Los elementos con display:inline solo ocupan el ancho necesario y no aceptan propiedades de tamaño como width o height

Display: Inline-Block

Los elementos con display:inline-block combinan las características de inline(no rompen línea) con block (aceptan width y height)
